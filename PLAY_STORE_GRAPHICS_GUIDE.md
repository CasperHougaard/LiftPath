# Google Play Store Graphics Guide for LiftPath

## Required Graphics

### 1. App Icon
- **Size:** 512 x 512 pixels
- **Format:** PNG (32-bit with alpha channel)
- **Requirements:**
  - No transparency (must have solid background)
  - Square format
  - High quality, sharp edges
  - Should match your app's launcher icon design

### 2. Feature Graphic (Banner)
- **Size:** 1024 x 500 pixels
- **Format:** PNG or JPG
- **Requirements:**
  - No rounded corners
  - No text overlay (Google adds app name automatically)
  - Should represent your app visually

### 3. Screenshots
- **Phone:** At least 2, up to 8 screenshots
  - Minimum: 320px height
  - Maximum: 3840px height
  - Aspect ratio: 16:9 or 9:16
- **Tablet (optional):** 7" and 10" tablet screenshots
  - Same requirements as phone

## Design Recommendations

### App Icon (512x512)
Based on your current design:
- **Background:** Solid color (green #4CAF50 or gradient)
- **Foreground:** Dumbbell/weight icon (white or contrasting color)
- **Style:** Clean, modern, recognizable at small sizes

### Feature Graphic (1024x500)
Suggestions:
- **Left side:** App icon or dumbbell illustration
- **Right side:** Key features or app interface preview
- **Colors:** Use your brand colors (#4CAF50 green, #FF9800 orange)
- **Text:** Minimal or none (Google adds app name)

## Tools to Create Graphics

### Free Options:
1. **Canva** (canva.com)
   - Templates for Play Store graphics
   - Easy to use, drag-and-drop

2. **Figma** (figma.com)
   - Professional design tool
   - Free for personal use

3. **GIMP** (gimp.org)
   - Free image editor
   - Similar to Photoshop

4. **Android Asset Studio** (online)
   - Can generate app icons from existing images

### Quick Steps:
1. Export your current launcher icon at high resolution
2. Use design tool to create 512x512 version with solid background
3. Create feature graphic showing app features
4. Take screenshots from your app running on device/emulator

## Color Palette
- Primary Green: #4CAF50
- Primary Dark: #388E3C
- Accent Orange: #FF9800
- Background: #F0F0F0
- Text Primary: #212121
